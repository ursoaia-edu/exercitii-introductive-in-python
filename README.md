# Exerciții introductive în Python

1. Realizați un program care citește de la tastatură un șir de caractere și afișează de câte ori apare litera "a" în text.
2. Cere utilizatorului numele și afișează-l într-un mesaj personalizat.
3. Citește două numere întregi și afișează suma lor.
4. Calculează aria unui dreptunghi folosind lățimea și lungimea introduse de utilizator.
5. Determină dacă un număr este par sau impar.
6. Transformă temperatura din Celsius în Fahrenheit.
7. Afișează primul și ultimul caracter dintr-un șir introdus de utilizator.
8. Inversează un șir de caractere.
9. Numără vocalele dintr-un cuvânt introdus.
10. Verifică dacă un număr este pozitiv, negativ sau zero.
11. Calculează media a trei numere.
12. Determină cea mai mare valoare dintre trei numere.
13. Creează o listă cu cinci elemente și afișeaz-o.
14. Adaugă un element nou într-o listă.
15. Elimină un element specific dintr-o listă.
16. Sortează o listă de numere.
17. Verifică dacă un element există într-o listă.
18. Creează un tuplu cu 3 valori și afișează-l.
19. Afișează lungimea unui tuplu.
20. Transformă un tuplu într-o listă.
21. Creează un set și adaugă câteva elemente unice.
22. Verifică apartenența unui element într-un set.
23. Creează un dicționar cu cheile 'nume', 'vârstă', 'oraș'.
24. Accesează valoarea unei chei dintr-un dicționar.
25. Adaugă o pereche nouă cheie-valoare într-un dicționar.
26. Calculează lungimea unui șir fără a folosi len().
27. Convertește un șir în majuscule și minuscule.
28. Verifică dacă un șir este palindrom.
29. Calculează factorialul unui număr folosind o buclă for.
30. Calculează suma numerelor de la 1 la n.
31. Afișează toate numerele pare dintr-un interval dat.
32. Afișează elementele unei liste folosind un for.
33. Folosește un while pentru a afișa numere de la 1 la 10.
34. Creează o funcție care primește două numere și returnează produsul lor.
35. Creează o funcție care verifică dacă un număr este prim.
36. Creează o funcție care returnează numărul de litere mari dintr-un șir.
37. Folosește try-except pentru a prinde o eroare de împărțire la zero.
38. Creează o listă de tuple (nume, vârstă) și sorteaz-o după vârstă.
39. Afișează valorile unui dicționar folosind o buclă for.
40. Creează o funcție recursivă care calculează factorialul.
41. Verifică dacă un an este bisect.
42. Calculează suma cifrelor unui număr.
43. Creează o listă de pătrate pentru numerele de la 1 la 10.
44. Creează un program care numără câte cuvinte are o propoziție.
45. Afișează tabelul de înmulțire pentru un număr dat.
46. Înlocuiește un caracter specific dintr-un șir cu altul.
47. Creează un program care calculează vârsta pe baza anului de naștere.
48. Folosește o expresie condițională pentru a decide dacă un elev a promovat.
49. Creează o listă cu 10 numere și afișează doar cele mai mari decât 5.
50. Scrie un program care calculează suma valorilor pare dintr-o listă.
